# 📅 DayGrid

A productivity tracking application that helps students compare planned work with actual work, identify procrastination patterns, track productivity gaps, and generate weekly performance insights.

Unlike traditional to-do apps, DayGrid focuses on accountability by highlighting missed tasks, unplanned time gaps, and recurring excuses that impact productivity.

---

## 🚀 Features

### 📊 Productivity Tracking

* Plan daily activities using time blocks
* Track completion percentage for each task
* Compare planned vs actual performance
* Monitor daily and weekly productivity trends

### ⏳ Gap Detection

* Automatically identifies unplanned time periods
* Categorize gaps as:

  * Procrastination
  * Scrolling
  * Rest
  * Other activities

### 📈 Analytics Dashboard

* Weekly productivity charts
* Progress tracking
* Productivity pattern analysis
* Planned vs actual comparison reports

### 🎯 Accountability System

* Excuse tracking for missed tasks
* Personal excuse database
* Weekly reflection reports
* Productivity insights and recommendations

### 🔥 Streak System

* Daily consistency tracking
* Achievement milestones
* Long-term habit formation support

### 📝 Weekly Reflection

* Best performing day
* Lowest productivity day
* Most productive time period
* Most common excuse
* Weekly improvement summary

### 📚 Student-Friendly Features

* Exam Mode
* Mood Tracking
* Goal Management
* Local data persistence

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/virexan/daygrid.git
```

### Navigate to Project

```bash
cd daygrid
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm start
```

Application runs at:

```text
http://localhost:3000
```
